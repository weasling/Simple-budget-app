## Endpoints

### Users Table
Used to store user information.

1. **Create User**:
Endpoint: `/data`

**Method**: POST  
**Headers**: `Content-Type: application/json`  
**Description**: Creates a new user

**Example Operations**:

Create User Anonymously in the background:
```json
{
    "app_id": "76efa9de-8130-4316-80d0-7ef758902073",
    "table_name": "users",
    "data": {
        "provider": "anonymous"
    }
}
```

**Response**
```json
{"id": "c77123fb-4c13-4896-89ad-7448f1f94d9a",  "provider": "anonymous", "created_at": "2025-04-20T04:57:10.976616+00:00"}
```
- Save the `id` as user_id, and use it as `user_id` for further operations, change status to logged in.
